## Installation Instructions
1. Clone the repository:
```   
git clone https://github.com/M-AdanAli/JavaFx-Media-Player.git

```
3. Open the project in your IDE (IntelliJ IDEA, Eclipse, etc.).
4. Ensure your environment has Java 25 and JavaFX SDK correctly configured.
5. Build and run the `MainApplication` class.

## Usage Instructions
- Use the **File > Open** menu to select supported video files.
- Playback controls located at the bottom:
- Use ‚ñ∂ / ‚è∏ buttons to toggle play/pause.
- Use ‚è™ / ‚è© buttons to rewind or fast forward 5 seconds.
